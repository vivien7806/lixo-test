EmployeeApi

An API for managing employees and their vacations.

Resources

  • POST /employee: create a new employee with the provided first and last name in the request body
  • GET /employee: retrieve a list of all employees
  • POST /employee/:id/vacation: add a new vacation for the employee with the specified id in the request path, with the start date, end date, and comment in the request body
  • GET /employee/:id: retrieve details for the employee with the specified id in the request path
  • POST /currentlyoff: retrieve a list of employees who are currently on vacation within the specified start and end dates in the request body

Request bodies

POST /employee

  
  {
    "firstName": "string",
    "lastName": "string"
  }

POST /employee/:id/vacation

  
  {
    "startDate": "YYYY-MM-DD",
    "endDate": "YYYY-MM-DD",
    "comment": "string"
  }

POST /currentlyoff

  
  {
    "startDate": "YYYY-MM-DD",
    "endDate": "YYYY-MM-DD"
  }
